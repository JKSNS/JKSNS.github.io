## Security Researcher
**Independent & Academic**
*2022 – Present*

I conduct independent security research focused on vulnerability assessment, security tooling development, and applied offensive security techniques.

This work spans web application security testing, network infrastructure assessment, and systems-level security analysis. I approach research methodically, combining automated scanning with manual testing to identify vulnerabilities that tools alone would miss.

On the tooling side, I develop automation scripts and utilities that accelerate common security workflows. These range from system hardening scripts that enforce secure baselines to reconnaissance tools that streamline the enumeration phase of assessments.

I contribute to open-source security projects and participate actively in the cybersecurity community, sharing knowledge through competition mentoring and collaborative research. I believe security knowledge improves when it is shared openly and tested rigorously.

My research interests include malware analysis, reverse engineering, and exploit development. I study these topics not to build offensive capabilities for their own sake, but because understanding how attacks work is essential for building effective defenses.

This independent work complements my competition experience by allowing deeper exploration of topics that competition time constraints do not permit. It is where curiosity drives the work, and the results feed back into better defensive capabilities.
